#Drone-detail

Repository for page "drone-detail"
You can see in real-time the status of your personal domotic drone.
When it is online, you can see in real-time the camera of your drone!

"Stanza": you can see where is your drone.

"Oggetto": If your drone transport an object, it result occupied, if don't has any object it is free!

"Connessione": you can control the status of the connection of your drone.

"Batteria": you can control the drone's battery percentual.

From this page you can command your drone for: change room
     release object
     connect on/off
     take it to rest (in charge)

You can surf to other page in the menù.
